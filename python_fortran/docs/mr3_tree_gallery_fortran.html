No Python re-implementation involved in producing the numbers — each kernel writes one log line per successful child, reconstructing diag(N·G·NT) in quad precision via libquadmath. The Python harness only parses logs.

The header flags matrices where BLK and NB diverge by more than 100× — those expose the difference between the two factorization regimes.
