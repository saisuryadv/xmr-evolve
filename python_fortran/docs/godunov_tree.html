Why does Godunov fail without the fix?

The bidiagonal has off-diagonals alternating between e_large = 0.999 and e_tiny = 2.5e-7. After building T_GK (size 40), the off-diagonals repeat with period 4. Standard multiplicative perturbation E(i) *= (1 + r*8*eps) changes the tiny entries by only ~5e-22 in absolute terms. When MR3 shifts into the cluster, the block factorization recurrence passes through E^2 at the tiny entries, which resets all accumulated variation. The shifted representation (GPLUS) becomes an arithmetic progression with constant increment per period — making FP starting vectors for different eigenvalues structurally identical.

How the fix works

Our 9-line patch in dlaxre_gk.f applies additive perturbation E(i) += r*FAC*EMAX to entries where |E(i)| < 0.1*EMAX and FAC*EMAX < |E(i)|. This gives the tiny entries ~1.8e-15 absolute change instead of ~5e-22 — enough to break the periodicity in the shifted representation. The eigenvalues become distinguishable as singletons at the root, eliminating the need for deep tree recursion.
